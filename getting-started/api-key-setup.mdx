## Genel Bakış

Platform API Anahtarı, AI Çağrı Merkezi Platformu'nun iletişim kurması için gerekli güvenlik anahtarıdır. Bu anahtar olmadan platformun AI özellikleri çalışmayacaktır.

## Platform API Anahtarı Nedir?

Platform API Anahtarı, şirketinizin:

- AI sesli asistanları başlatmasını
- Çağrı kayıtları ve transkriptler almasını
- Gerçek zamanlı çağrı analizleri yapmasını
- Otomatik ticket oluşturmasını sağlar

<Info>
  **Önemli**: Platform API anahtarı kurulumu otomatik olarak yapılmaktadır. Sistemimiz, hesabınız oluşturulduğunda gerekli API yapılandırmasını otomatik olarak gerçekleştirir.
</Info>

## İlk Kurulum Sırasında API Yapılandırması

İlk kez giriş yaptığınızda **Kurulum Sihirbazı** sizi karşılayacak:

### Adım 1: Organizasyon Kurulumu

1. Şirket bilgilerinizi doldurun
2. **Kurulumu Tamamla** butonuna tıklayın

### Adım 2: Platform API Yapılandırması

Platform API anahtarı otomatik olarak yapılandırılır. Herhangi bir işlem yapmanıza gerek yoktur.

<Note>
  Sistem, hesabınız için gerekli API bağlantılarını otomatik olarak kurar ve test eder.
</Note>

## API Bağlantısını Kontrol Etme

### Adım 1: Ayarlar Sayfasına Gidin

1. Sol menüden **Ayarlar**'ı seçin
2. **Organizasyon** sekmesinin alt kısmındaki **API Anahtarı** bölümüne gidin

### Adım 2: Bağlantı Durumunu Kontrol Edin

API bağlantı durumunu kontrol panelinde görebilirsiniz:

- **Bağlı** (Yeşil): API bağlantısı aktif ve çalışıyor
- **Bağlı Değil** (Kırmızı): Bağlantı sorunu var
- **Bilinmiyor** (Sarı): Durum test edilmedi

## API Bağlantı Durumu Yönetimi

### Bağlantı Durumunu Görüntüleme

**Ayarlar > Organizasyon** sekmesinde:

- API bağlantı durumunu görebilirsiniz
- Bağlantı durumunu (Bağlı/Bağlı Değil/Bilinmiyor) kontrol edebilirsiniz

### Bağlantıyı Test Etme

1. **Bağlantıyı Test Et** butonuna tıklayın
2. Sistem API bağlantısını doğrulayacak
3. Sonuç mesajını kontrol edin:
   - **"API bağlantısı başarılı"**: Sistem çalışıyor
   - **"API bağlantısı başarısız"**: Destek ekibi ile iletişime geçin

## API Bağlantı Uyarıları

### Uyarı Bannerı

API bağlantınız sorunlu ise:

- Sayfanın üst kısmında **uyarı bannerı** görünecek
- "API bağlantısı kurulamadı" mesajı alacaksınız
- Destek ekibi ile iletişime geçmeniz önerilir

### Bağlantı Durumu Kontrolü

**Kontrol Paneli**'nde AI Entegrasyonu kartı:

- **Bağlı** (Yeşil): API bağlantısı çalışıyor
- **Kısmen Bağlı** (Sarı): Bazı sorunlar var
- **Bağlı Değil** (Kırmızı): API bağlantısı kurulamadı

## Güvenlik

### Platform Güvenliği

<Tip>
  - ✅ Tüm API bağlantıları şifreli (HTTPS) üzerinden yapılır
  - ✅ API yapılandırması platformda güvenli olarak saklanır
  - ✅ Sadece yetkili kullanıcılar API ayarlarına erişebilir

## Sık Sorulan Sorular

<AccordionGroup>
  <Accordion title="API yapılandırması için bir şey yapmam gerekiyor mu?">
    Hayır, API yapılandırması otomatik olarak yapılır. Herhangi bir işlem yapmanıza gerek yoktur.
  </Accordion>

  <Accordion title="API bağlantısı neden 'Bağlı Değil' görünüyor?">
    Geçici bir sistem sorunu veya ağ bağlantısı problemi olabilir. Bağlantıyı test edin ve sorun devam ederse destek ekibi ile iletişime geçin.
  </Accordion>

  <Accordion title="API kullanımının maliyeti var mı?">
    Platform kullanımı kredi sistemi üzerinden yönetilir. API bağlantısının kendisi için ek ücret yoktur.
  </Accordion>

  <Accordion title="API bağlantısı kesilirse çağrılarım ne olur?">
    Aktif çağrılar tamamlanana kadar devam eder. Yeni çağrılar başlatılamaz. Bağlantı restore edildiğinde normal operasyonlara devam edilir.
  </Accordion>
</AccordionGroup>
