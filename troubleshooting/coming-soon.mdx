## Hızlı Yardım

Yaygın sorunlar için şu bölümlere bakabilirsiniz:

### API Anahtarı Sorunları
API anahtarı ile ilgili sorunlar için [API Anahtarı Kurulumu](/getting-started/api-key-setup) sayfasındaki sorun giderme bölümüne bakın.

### SIP Trunk ve Telefon Sorunları
Telefon bağlantısı sorunları için [SIP Trunk Kurulumu](/phone-numbers/sip-trunk-setup) sayfasına göz atın.

### AI Agent Sorunları
Agent oluşturma ve yapılandırma sorunları için [AI Agent Oluşturma](/voice-ai-agents/create-agent) sayfasını inceleyin.

## Yardım ve Destek
